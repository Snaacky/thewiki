# Deluge

Deluge is a free, open-source, and cross-platform BitTorrent client. It is often used by seedboxes for racing and works as both a standalone client and a thin client for a remote server.

# Installation

  1. Download the latest .exe version of Deluge
  2. Run the installer and follow the on-screen instructions to complete installation
  1. Download the latest .dmg version of Deluge
  2. Run the installer and follow the on-screen instructions to complete installation
  1. Download the latest version of Deluge for your distro
  2. Run the installer and follow the on-screen instructions to complete installation

# Configuration

# ltconfig

ltConfig is a popular plugin used for racing torrents on seedboxes. It modifies the libtorrent settings directly and comes with presets.

  1. Download the correct version of ltConfig for your version of Deluge:
    • Deluge v2.x or later: v2.0.0
    • Deluge v1.x: v0.3.1
  2. Launch Deluge. In the top bar, go to Edit -> Preferences
  3. Under Plugins, click on Install and select the ltConfig file
  4. Enable ltConfig from your plugin list and hit Apply
  5. Under ltConfig, check Apply settings on startup. For racing, select the High Performance Seed preset -> Load preset -> Apply
